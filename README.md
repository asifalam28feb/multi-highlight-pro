# Multi-Highlight Pro

A modern, Manifest V3 multi-word highlighter.

## Installation for Team Members
1.  **Download as ZIP:** Click the green `<> Code` button and select "Download ZIP".
2.  **Unpack:** Unzip the downloaded file.
3.  **Chrome Extensions:** Open Chrome and navigate to `chrome://extensions`.
4.  **Developer Mode:** Toggle **"Developer mode"** in the top-right corner to **ON**.
5.  **Load:** Click **"Load unpacked"** and select the folder you unzipped.

## Important Note on Security and Testing
This extension is designed for public websites and general testing. Modern Chrome security policies **intentionally block unpacked extensions from running in high-security environments.**

### Where it WILL NOT work:
* Internal corporate domains (e.g., `corp.google.com`).
* Chrome system pages (e.g., `chrome://settings`).
* Google search results pages (blocked by modern privacy rules).
* Password entry forms and bank websites (high-security zones).

### Where it WILL work:
* Standard public `https` websites.
* Internal team documentation sites (unless on a `corp` subdomain).
* Public knowledge bases (like Wikipedia).

**For reliable testing, do not test on Google or internal corporate pages. Please test on a public news site or a different non-Google website.**
